Criando Parágrafos e Quebra De Linhas!

Posso utilizar a tag <br> para quebrar linhas em meu documento html

Em meu documento html, posso usar a tag <p> para criar parágrafos.

  Outra coisa que posso fazer, é, colocar espaço e inserir caracteres como, por exemplo: ® ™ € © £ ¢ ¥ ∅ ∑ δ ← ↑ → ↓

E para fechar com chave de ouro, também é possível inserir emojis 🤓 🖖 🚀
